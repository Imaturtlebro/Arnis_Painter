Arnis is a free, open-source tool developed by Louis Erbkamm that converts real-world geographic locations into detailed, playable Minecraft maps. By utilizing data from OpenStreetMap for structures and AWS Terrain Tiles for elevation, the program generates block-based replicas of neighborhoods and cities for both Java and Bedrock editions. Originally written in Python and later ported to Rust for better performance, the application has gained significant traction within the gaming community and academic circles. Researchers have even utilized the tool for educational projects like Floodcraft, which teaches students about flood mitigation through interactive simulations. Users can easily create these worlds by selecting a region on a digital map, allowing for personalized exploration of actual landmarks and topography. This modular software continues to evolve with updates that improve architectural realism and provide browser-based generation options.
